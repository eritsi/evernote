# 麻婆茄子

\

![Evernote Camera Roll 20161206 085258.jpg](./images/Evernote%20Camera%20Roll%2020161206%20085258.jpg)

\

茄子3本

縦切り斜め1cm切り

\

中華鍋を火にかけながらサラダ油、にんにくスライス半分、しょうが1cm角薄切り半分、豚ひき肉100g、茄子を炒めていく。サラダ油足して、茄子に染ませる。

\

残りの半分のしょうが、にんにくを鍋にいれて蓋する

麻辣醤を大きなスプーン1.5杯いれ、水を150ccくらいいれまぜる。

お砂糖軽く一杯ちょっと

塩は調整で入れる、水60cc追加

茄子は茹でるのに時間がかかるので10分くらいは煮るイメージ

小口切りネギを入れたらすぐ水溶き片栗粉回し入れ、30秒くらいでお皿に盛る

# チーズハンバーグ　ポルチーニクリームソース

\

![69b9ce575312.jpg](./images/69b9ce575312.jpg)

ハンバーグ（合挽き）
150g×2個

オリーブオイル
大さじ1

ドライポルチーニ
5g

ぬるま湯
150ml

生クリーム
1本（100ml）

オリーブオイル
小さじ1

にんにく
1/2片程度

バター
1個（8g）

とろけるチーズ
2袋（40g）

クレソン
2本

##### 

##### 

##### 〜ハンバーグの準備をします〜

ドライポルチーニをぬるま湯（150ml）で5分程度戻しておく。

POINT

戻し汁とポルチーニを分けておく。

にんにくをみじん切りにする。

クレソンは冷水にさらす。

![0bad47f9df5b.jpg](./images/0bad47f9df5b.jpg)

##### 〜ハンバーグを作り始めます〜

ハンバーグを俵型に形を整える。

POINT

手のひらでキャッチボールする要領で空気を抜きます。手で叩くように真ん中から外側に伸ばし中央にくぼみを作ります。

![5a3b972b909e.jpg](./images/5a3b972b909e.jpg)

フライパンにオリーブオイル（大さじ1）を入れ、中火にかける。熱したフライパンにハンバーグを入れて、両面が色づくまで焼く。バットにとり、温かい場所に置く。

POINT

ハンバーグはあとで蒸し焼きにするので、表面に軽く焦げ目をつけるだけで大丈夫です。

![cf70efb37593.jpg](./images/cf70efb37593.jpg)
![30c956a985ab.jpg](./images/30c956a985ab.jpg)

同じフライパンをキッチンペーパーでキレイに拭いて、オリーブオイル（小さじ1）とにんにくを入れ、中火にかける。

![c45511c2ee84.jpg](./images/c45511c2ee84.jpg)

にんにくの香りが立ったらポルチーニ（汁は入れない）を加え炒める。軽くにんにくと混ぜ合わせたら、ポルチーニの戻し汁と生クリーム（1本）を加え、一煮立ちさせる。

![aa1c41471b03.jpg](./images/aa1c41471b03.jpg)
![f695856a4ccf.jpg](./images/f695856a4ccf.jpg)
![9790e786bb00.jpg](./images/9790e786bb00.jpg)

ハンバーグをソースのフライパンに戻し、蓋をして3〜4分程蒸し焼きにして完全に火を入れる。

POINT

竹串を刺して、肉汁が透明になったら火が入っています。

![0c57924ee5a8.jpg](./images/0c57924ee5a8.jpg)
![7f8402bf851c.jpg](./images/7f8402bf851c.jpg)
![101ef1138985.jpg](./images/101ef1138985.jpg)

\

##### 〜チーズハンバーグを完成させます〜

仕上げにハンバーグの上にチーズを乗せ、蓋をして余熱でチーズが溶けるように蒸らす。

![ade860763de5.jpg](./images/ade860763de5.jpg)

皿にハンバーグを盛る。

フライパンのソースをお好みの濃度に煮詰め、仕上げにバター（1個）を加える。

![6fae361fdcdb.jpg](./images/6fae361fdcdb.jpg)
![b4c4bc3e353b.jpg](./images/b4c4bc3e353b.jpg)

ハンバーグにソースをかけ、クレソンを飾って完成。

![78d84dbb231c.jpg](./images/78d84dbb231c.jpg)

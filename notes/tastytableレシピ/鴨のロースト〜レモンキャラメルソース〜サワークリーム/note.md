# 鴨のロースト〜レモンキャラメルソース〜サワークリームのマッシュポテト添え

\

![d7d736a991c8.jpg](./images/d7d736a991c8.jpg)

鴨ロース
1枚

塩
小さじ1/2と小さじ1/3

胡椒
適量

イタリアンパセリ
2g程度

塩
ひとつまみ

＜レモンキャラメルソース＞
-

レモン
1個

砂糖
70g（目安：大さじ8弱）

水
大さじ2

＜マッシュポテト＞
-

サワークリーム
1個（90ml）

じゃがいも（メークイン）
2個

オリーブイオル
小さじ1

塩
ひとつまみ

黒胡椒
適量

##### 〜鴨のローストの準備をします〜

・レモンの中心を薄く2枚スライスする。（飾り用）

・残りは絞ってレモン汁にする。

・じゃがいもの皮をむき、半分に切って水にさらす。

![24493c6bff02.jpg](./images/24493c6bff02.jpg)

1個ずつラップに包み、まとめて500Wの電子レンジで6分加熱する。

POINT

1個ずつ包むことで、じゃがいもに均等に火が入ります。

![3466b97be9af.jpg](./images/3466b97be9af.jpg)

鴨の身の筋（白い部分）を外し、皮目に格子状に包丁を入れる。

![a682fb61eef9.jpg](./images/a682fb61eef9.jpg)
![8a6f86ba6bcf.jpg](./images/8a6f86ba6bcf.jpg)
![39e30a514daf.jpg](./images/39e30a514daf.jpg)

皮目に塩（小さじ1/2）、胡椒（適量）、身側に塩（小さじ1/3）をする。

POINT

皮目は脂で流れやすいので、少し多めに塩をしてください。

![f1201d4bf83d.jpg](./images/f1201d4bf83d.jpg)

##### 〜煮ている間に鴨のソースと付け合わせを作ります〜

鍋に砂糖（70g）を入れて中火にかけて、焦がしてキャラメルを作る。

POINT

砂糖が溶け始め、キャラメル色になるまで火を入れてください。

POINT

途中鍋を揺らしながら、砂糖を焦がしすぎないよう気を付けてください。

![861bba4103bd.jpg](./images/861bba4103bd.jpg)
![88e148b2491b.jpg](./images/88e148b2491b.jpg)
![5ad3cd0220d5.jpg](./images/5ad3cd0220d5.jpg)

砂糖が溶け、キャラメル色になったら鍋を火から離して、水（大さじ2）を加えよく混ぜ、さらにレモン汁を加え混ぜ合わせる。

![954bf3c4a132.jpg](./images/954bf3c4a132.jpg)

再度中火にかけて2/3程度になるまで煮詰める。

POINT

冷めるとソースが少し硬くなるので、ソースにスプーンで線を書いて、鍋底の跡がすぐに消える程度が煮詰めあがりの目安です。

POINT

火を入れすぎ、ソースが固まってしまった場合は、水（分量外）を足し火にかけ、ソースを溶かすようにしてください。

![e24bc0523859.jpg](./images/e24bc0523859.jpg)
![8935fb0f71f5.jpg](./images/8935fb0f71f5.jpg)

加熱したじゃがいもをボウルに入れ、ざっくりじゃがいもの食感が残るくらいに潰す。オリーブオイル（小さじ1）と塩（ひとつまみ）をする。

![36df5e38a9ab.jpg](./images/36df5e38a9ab.jpg)

サワークリーム（2/3個）を加えてさらに混ぜ、マッシュポテトを作る。

![ad4cf6e09fee.jpg](./images/ad4cf6e09fee.jpg)
![52bac754ed03.jpg](./images/52bac754ed03.jpg)

\

##### 〜鴨のローストを完成させます〜

皿にマッシュポテトを盛り付け、残りのサワークリームをのせて、黒胡椒（適量）を上に振る。

鴨をスライスして皿に盛り付け、軽く塩（ひとつまみ）をする。お好みの量のソースをかけて、スライスしたレモンとイタリアンパセリを飾る。

![7f2f97ea0ecf.jpg](./images/7f2f97ea0ecf.jpg)

# 自家製スペアリブ　いんげん豆添え＆ワンタン鍋　３種のタレで

**50min※**

豚肉を漬ける時間30分を除く

![Attachment-79ca2821.jpg](./images/Attachment-79ca2821.jpg)

**Cooking description**

**- 料理説明 -**

ご家庭でできる本格スペアリブを叉焼にアレンジしました。骨付きでボリュームがあり、炊きたての白米やビールのおつまみににもぴったりな一品です。スパイスと煮込んだ少し甘めの白いんげん豆のアクセントが絶妙です。是非豪快にお召し上がりください。

副菜にはつるりとのどごしの良いワンタンを鍋仕立てにしました。３種類のつけダレをかえながら様々な味をお楽しみ下さい。

残った出汁にご自宅の麺を入れ、タレとあえれば汁なし担々麺の出来上がり。アレンジもお楽しみいただける一品です。

##### 〜スペアリブを焼きます〜

豚スペアリブを200度に温めたオーブンで20分焼く。

\

![Attachment-0abbc389.jpg](./images/Attachment-0abbc389.jpg)

途中10分経ったらひっくり返す。

\

##### 〜焼いている間にワンタン鍋を仕上げます〜

鍋に水（1000ml）と鶏がらスープを入れ、沸騰させる。

\

鶏がらスープはひとつまみだけ残しておいてください。

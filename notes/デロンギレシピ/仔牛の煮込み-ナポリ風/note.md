# 仔牛の煮込み ナポリ風

### Neapolitan Stewed Veal仔牛の煮込み ナポリ風

ナポリの家庭料理です。仔牛でなくとも豚フィレ肉で代用できます。また、グリーンピースでなくても、空豆や乾燥豆を使っても素朴な味わい。

#### 作り方

1.　仔牛肉はーロ大に切って、軽く塩、こしょう、ジンジャーパウダーを振っておく。\
\
2.　鍋にオリーブオイルとにんにくを入れて弱火にかけ、香りが立つまで時間をかけてゆっくり炒める。セロリと人参のみじん切りを加えさっと炒める。\
\
3.　ここに(1)を入れて色が変わったらグリーンピースを加え、さらに混ぜながら炒める。\
\
4.　粗切りのバジルとトマトをつぶしながら加え、缶汁も入れて、軽く塩、こしょうをし、鍋の蓋をして200度のオーブンで20～30分煮る。\
\
5.　仕上げの味を整え、皿に盛って飾り用(A)とバージンオリーブオイルをまわしかける。

##### ![head\_point\_1338711213\_437222.gif](./images/head\_point\_1338711213\_437222.gif)

・ 冷凍の豆はこの料理にはむきません。\
\
・ バジルは水洗いして切ったまま放置しておくと茶色に変色しますので、飾る直前に切ります。\
\
・ この煮込みは、本来50～60分火にかけて煮つづけるものですが、スープを加えないため焦げやすいのが難点でした。オーブンならば、それがうまくいきます。熱の輪の中に鍋を置き、じっくりと煮汁の蒸発を防ぎながら煮てください。肉も身やせしません。\
\
・ おでん、シチュー、コンニャクの煮込み等もオーブンは得意です。ぜひ試してください。

#### 【下準備】

・ にんにくは厚めにスライスする。\
・ パプリカは縦半分に切り種をとり、1cm幅で切る。\
・ エリンギは横半分に切り縦に5mm程にスライスする。\
・ 乾燥ポルチーニ茸は100ccの水にもどす。\
・ パルミジャーノはすりおろす。

#### 作り方

1.　フライパンにオリーブオイルをひきにんにくを入れ火をつける。\
\
2.　にんにくの香りがしてきたらパプリカを入れ中火にし、しんなりするまで炒める。\
\
3.　しんなりしてきたら塩、こしょうをし、エリンギを入れ８分程炒める。\
\
4.　強火にし乾燥ポルチーニ茸をもどし汁もいっしょに入れ煮詰める。\
\
5.　煮詰まってきたら中火に戻しトマトペーストを入れかるく炒める。\
\
6.　弱火にし生クリームを入れ、とろみが出るまで炒める。\
\
7.　パスタをゆでる。\
\
8.　6にとろみが出てきたらパルミジャーノで味を調える。\
\
9.　ゆであがったパスタを加えて、全体をからめ、火を止める。

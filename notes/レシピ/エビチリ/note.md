# エビチリ

## えびのチリソース

#### 

(4人分) \
・芝えび（むき身）200g\
----------\
【えびの下味】\
・塩 小さじ1/4　\
・こしょう 少々\
・卵白 1/4コ分\
・かたくり粉 大さじ1+1/2\
・サラダ油 小さじ1\
----------\
・グリンピース 40g\
・にんにく 大さじ1/2\
（みじん切り）　\
・しょうが 大さじ1\
（みじん切り）\
----------　\
・豆板醤（トーバンジャン）\
　　　　　　　　　　大さじ1　\
・水 少々\
----------\
・トマトケチャップ 大さじ2\
・スープ＊ カップ1/2\
・ねぎ（みじん切り）\
　　　　　　 1/4本分（40g）\
・卵 1コ＊＊\
（サラダ油・酒・砂糖・塩\
　・こしょう・かたくり粉・酢）\
\
＊顆粒チキンスープの素（中国風）を表示どおりに湯で溶く。\
＊＊えびの下味の卵の残りでよい。\
\
※辛くしたいときは、一味とうがらし・ラーユ各適量を加える。

#### 

《プロのコツ①》\
●えびの下味は卵白とかたくり粉で包み込み、うまみを閉じ込める。\
\
水分を吸いにくいえびは、水けを紙タオルでふき取ってから下味をつける。えびに塩、こしょうをもみこみ、よく混ぜた卵白を加えて軽く泡立つまで混ぜる。かたくり粉を加えて全体に水けがなくなるまで混ぜ、サラダ油を加えてサックリと混ぜる。\
\
《プロのコツ②》\
●えびはあらかじめ多めの油でいためておき、プリプリの食感に。\
\
油を熱したら一度火を止めてえびを並べ入れる。再び火をつけ、えびの表面の衣が固まり、八分どおりに火が通るまで両面を焼く。衣がえびの水分とうまみを閉じ込めてプリプリの食感になる。\
\
《プロのコツ③》\
●豆板醤と香味野菜は、弱火でゆっくりといため、香りをしっかりと引き出す。えびを加えたら手早く仕上げる。\
\
A.　にんにく、しょうが、豆板醤、ケチャップを弱火で混ぜながらいためる。\
\
B.　透明な赤い油が分離して出てくるまで、焦がさないようにいため、香りを引き出す。\
\
C.　スープと残りの調味料を加え、ボコボコと煮立ってきたらえびとグリンピースを加える。\
\
D.　ねぎを加えたらすぐに水溶きかたくり粉を少しずつ注ぎ入れ、サックリと混ぜる。\
\
E.　最後に溶き卵を回し入れるとチリソースがフワッと柔らかく仕上がる。\
\
F.　つや出しの油と、油をさっぱりとさせるために酢を加えて、サックリと混ぜる。

#### 

1.　えびは背ワタがあれば取ってよく洗い、水けをしっかりとふいてボウルに入れ、下味の材料を順に加えて混ぜる（コツ①）。グリンピースはゆでて水にとる。\
\
2.　フッ素樹脂加工のフライパンにサラダ油大さじ2を熱し、えびを並べ入れ、両面を八分どおりに焼いて一度取り出す（コツ②）。\
\
3.　チリソースをつくる（コツ③）。2のフライパンにサラダ油大さじ2を熱し、にんにく、しょうが、水でのばした豆板醤、トマトケチャップを入れて、混ぜながら弱火でいためる。香りがたち、油の色が透明な赤になったらスープを加え、酒大さじ1/2、砂糖小さじ1、塩・こしょう各少々を加える。豆板醤は焦げないように、水でマヨネーズぐらいの柔らかさにのばしてからいためるとよい。\
\
4.　3がひと煮立ちしたら、2のえび、水けをきったグリンピースを加えてざっといためる。ねぎを入れ、かたくり粉大さじ1を同量の水で溶いて加え、とろみをつける。溶き卵を加え、ざっと混ぜ、サラダ油大さじ1と酢少々を加えてサックリと混ぜる。\
\
5.　器に盛り、熱いうちに食べる。

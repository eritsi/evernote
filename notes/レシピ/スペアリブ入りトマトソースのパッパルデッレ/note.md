# スペアリブ入りトマトソースのパッパルデッレ

### Pappardelle al sugo di carneスペアリブ入りトマトソースのパッパルデッレ

トマトソースは煮詰めれば煮詰めるほど美味しくなりますよ！

#### 【下準備】

・ 玉ねぎはみじん切りにし、にんにくはスライスする。\
・ 唐辛子は半分に切り種をとる。\
・ パルミジャーノはすりおろす。

#### 作り方

1.　鍋にオリーブオイル、玉ねぎ、にんにく、唐辛子を入れ、10分程弱火ですき通るまで炒めたらスペアリブを入れ、さらに20分炒める。\
\
2.　塩、こしょうをしトマト缶と水を入れ、強火で沸騰させる。\
\
3.　沸騰したらふたをし弱火で50分程煮込む。\
\
4.　パスタをゆでる。\
\
5.　パスタがゆであがったらざるで湯を切り、鍋に戻してソースの半分を入れ混ぜる。\
\
6.　パスタを皿に盛り合わせたら残りのソースをかけ、スペアリブをのせる。パルミジャーノをかけ出来上がり。

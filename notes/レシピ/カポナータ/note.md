# カポナータ

|  |  |
| --- | --- |
| ソース仕立てのカポナータ |  |

![a0056451\_15182982\_1338714204\_196020.jpg](./images/a0056451\_15182982\_1338714204\_196020.jpg)
\
\
夏野菜をたくさん使って、トマトで煮込んだカポナータです。\
フランス語ではラタトゥイユですね。\
具を小さめにしてソース仕立てにしてみました。\
\
\
\
**「ソース仕立てのカポナータ」**\
\
\
【材料】\
\
3〜4人分\
＊茄子大きめ1本\
＊ピーマン２個\
＊セロリ１本\
＊人参1/2本\
＊タマネギ1/2個\
＊ズッキーニ、パプリカなどお好みでプラスしてください。\
\
＊基本のトマトソース300cc\
＊オリーブ油大さじ2\
＊白ワイン50cc\
＊塩胡椒少々\
\
★材料はすべて1〜2cm角程度に切っておく。\
★基本のトマトソースの作り方は[こちら](http://aiaicafe.exblog.jp/2456756/)\
\
\
\
\
【作り方】\
\
\
1.　【炒める】\
　　鍋にオリーブ油を熱し、野菜を炒める。\
　　油が回ったら白ワインを回しかけてアルコール分を飛ばす。　　\
\
\
2.　【煮る】\
　　野菜がしんなりしたら基本のトマトソースを投入。\
　　蓋をして火を弱め、10分ほどコトコト煮ます。\
　　仕上げに味を見て塩胡椒で調節してできあがり！\
\
\
![a0056451\_152187\_1338714204\_209159.jpg](./images/a0056451\_152187\_1338714204\_209159.jpg)
\
もし、トマトソースが無い！って言う方は、\
ソースの代わりにトマトの水煮缶（400g入り）を使ってくださいね。\
その場合、手でトマトを潰して入れ、少し長めに煮込んだら\
最後に蓋を取って水分を飛ばします。\
\

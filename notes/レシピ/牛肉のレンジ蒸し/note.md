# 牛肉のレンジ蒸し

**#### 包んでうまみを閉じ込める**牛肉のレンジ蒸し**

|  |
| --- |
| ![5190\_0\_0.jpg](./images/5190\_0\_0.jpg) |

料理データ

|  |  |
| --- | --- |
| ■カロリー(１人分) | 334kcal |
| ■塩分 | 1.4g |
| ■調理時間 | 10分 |

##### 材料(２人分)

|  |  |  |
| --- | --- | --- |
| 牛こま切れ肉…150g |  | 長ねぎ…10cm |
| 下味(酒、しょうゆ各大さじ１、片栗粉小さじ１、粗びき黒こしょう少々) |  | ミニトマト…６個 |
| じゃがいも…１個 |  | 万能ねぎの小口切り…少々 |

##### 作り方

|  |  |  |
| --- | --- | --- |
| 1 | じゃがいもは四つ割りにして、２mm幅のいちょう切りにする。ねぎは斜め薄切りに、ミニトマトは縦半分に切る。 |  |
| 2 | ボウルに牛肉と、下味の材料を入れてよく混ぜる。１のじゃがいもと長ねぎを加えて、さらに混ぜる。 | ![5190\_402.jpg](./images/5190\_402.jpg) |
| 3 | 40cm長さのオーブン用ペーパーを２枚用意し、ミニトマト、じゃがいも、牛肉、長ねぎ、ミニトマトの順に半量ずつこんもりとのせる。手前と向こうの端を合わせて折りたたみ、さらに両側を下に折り込んで包み、耐熱皿にのせて電子レンジで約５分加熱する。ペーパーごと器に盛り、ペーパーを開いて万能ねぎを散らす。 | ![5190\_403.jpg](./images/5190\_403.jpg) |

# 海の幸のオーブン焼きパエリア風

#### 作り方

1.　パエリア鍋にオリーブオイルとたまねぎを入れしんなりするまで炒めます。次ににんにくを入れ香りが出たら小海老を加え、塩こしょうをふり全体を良く混ぜます。\
\
2.　1.にご飯を加えてよく混ぜ、ターメリックをふり入れご飯全体が黄色になるように炒めます。その時ご飯が冷たいと混ざりにくいので一度電子レンジで温めてからの方がパラパラになって混ざりやすくなります。\
\
3.　火からはずし、ご飯を平らにしてから野菜や海の幸を盛り合わせ塩少々とオリーブオイルを全体に振り、あらかじめ180℃に温めたオーブントースターで約5分間加熱します。\
\
※アルミホイルをかぶせて焼き色を調整する方法もあります。

# ティラミス

#### 作り方

1.　マスカルポーネに卵黄を加えゴムべらでしっかりと混ぜる。\
\
2.　生クリームにグラニュー糖を加え、8分立てにする。\
\
3.　(1)にグランマニエを混ぜこむ。\
\
4.　(3)を(2)に半分ずつ加え、なめらかなクリームを作る。\
\
5.　サヴォイアルディはバットに並べて、エスプレッソコーヒーを染み込ませておく。\
\
6.　サヴォイアルディの余分なコーヒーをペーパーで軽くぬぐっておき、キャセロールにクリーム、サヴォイアルディ、クリーム、サヴォイアルディ、クリームの順に組み立てる。\
\
7.　お皿に盛り付けた後、茶漉しでココアパウダーをふりかけて出来上がり。

##### ![head\_point\_1338712484\_024558.gif](./images/head\_point\_1338712484\_024558.gif)

・ サヴォイアルディは軽い焼き上がりのイタリアのフィンガービスケットです。シロップをよく吸収するのでしっとりとした仕上がりになり、ティラミスにぴったりです。手に入らない時はフィンガービスケットで代用してもかまいません。
